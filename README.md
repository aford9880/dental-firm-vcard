# DentalCare — Стоматологическая клиника

![DentalCare Screenshot](dental.png)

## О проекте

Современный одностраничный сайт стоматологической клиники с онлайн-записью через Telegram, картой и фотогалереей.

### Демо

[Посмотреть видео работы сайта](teeth.mp4)

### Функционал

- Hero с анимацией статистики
- Карточки услуг с ценами
- Команда врачей
- Галерея работ клиники
- Таблица цен
- Слайдер отзывов
- Форма записи → Telegram Bot API
- Карта Leaflet
- FAQ
- Полный адаптив

### Технологии

- HTML5 + CSS3
- JavaScript (vanilla)
- Leaflet (карта)
- Telegram Bot API
- Jest (21 тест)
