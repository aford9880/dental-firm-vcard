# Стоматология / Медицинская клиника — План ⭐⭐⭐

## Цель
Многостраничный или одностраничный сайт с доверительным дизайном, онлайн-записью и уведомлениями в Telegram.

## Структура
1. **Header** — лого, навигация (главная, услуги, врачи, цены, контакты), CTA
2. **Hero** — фото клиники/улыбка, заголовок, запись на приём
3. **Услуги** — карточки с ценой: терапия, хирургия, ортопедия, имплантация, ортодонтия
4. **Врачи** — карточки: фото, ФИО, специализация, стаж
5. **Цены** — таблица/список услуг с ценами
6. **Отзывы** — слайдер с фото пациентов
7. **Лицензии** — блок с сертификатами/лицензиями (сетка изображений)
8. **Форма онлайн-записи** — имя, телефон, услуга, дата/время → Telegram Bot API
9. **Контакты** — карта (Leaflet), адрес, телефон, режим работы
10. **Footer**

## Фишки
- Telegram Bot API для уведомлений о записи
- Блок лицензий — доверие
- Микроразметка JSON-LD (Organization, MedicalBusiness)

## Технологии
- HTML5 + CSS3 + JS
- Leaflet (карта)
- Telegram Bot API (уведомления)
- SEO + Open Graph + JSON-LD
